# typescript

## 环境搭建

* 步骤

  1. 下载安装node

     

  2. 安装`typescript`

     ````shell
     npm install typescript -g
     ````

  3. 创建ts文件

     

  4. 使用命令将ts文件编译为js文件

     1. `cmd`进入命令行

     2. 进入ts所在的文件目录

     3. 执行命令

        ````shell
        tsc xxxx.ts
        ````

* 类型的别名

  ````typescript
  // type关键词创建一个类型的别名  这样就可以多处使用
  
  type myType = 1 | 2 | 3 | 4 | 5
  
  let a1: myType
  
  let a2: myType
  ````
